ページ一覧

用語定義

システム開発における様々な用語・呼称を統一し、解釈のブレをなくします。

コーディング規約 / C#

.NET 標準を土台にした C# の命名、表記、初期化、コレクション、ツール、非同期、例外などの規約です。

基本アーキテクチャ

MVC、MVVM、レイヤード構成など、設計の軸となるアーキテクチャを解説します。

Unity セッティング

Unity の設定、アセット運用、Editor 拡張、ビルドまわりの基準です。

Visual Studio セッティング

フォーマット、分析、拡張機能、ソリューション構成の基準です。

テスト方針

ユニットテスト、PlayMode テスト、再発防止の考え方を解説します。

レビュー観点

命名、責務分割、例外、null、安全性、変更影響の確認観点です。

変更履歴

文書の更新履歴と、構成の変化を追跡します。
