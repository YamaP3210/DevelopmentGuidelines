EditorConfig

.editorconfig を基準に、インデント、改行、名前付け、コードスタイルを揃えます。改行は Visual Studio 標準の CRLF を基準にします。Visual Studio の個人設定だけに依存しません。

root = true

[*.cs]
indent_style = space
indent_size = 4
tab_width = 4
end_of_line = crlf
charset = utf-8
insert_final_newline = true
trim_trailing_whitespace = true

改行

テキストファイルは Visual Studio 標準の CRLF を基準にします。
